# Custom Data Retention Policies

**What it does:** Set how long different types of data are kept. Example: audit logs for 7 years, deleted customer records for 30 days then purged, messages for 1 year. Automatic cleanup. Legal hold overrides for litigation.

## Key Benefits

- **GDPR/CCPA compliance:** Proving you don't keep data forever
- **Reduced liability:** Less data = less to breach
- **Storage costs down:** Old data auto-purged
- **Legal hold flag:** Freeze deletion during litigation

## Real-World Example

**Real-World Example: "Istanbul Healthcare Network"**

Turkey's KVKK (personal data law) requires that patient data be deleted X days after account closure. Manually tracking this across 140,000 patient records was impossible.

With Zyrix Data Retention, policies run automatically: closed patient accounts → PII purged after 180 days. **KVKK compliance automated. Penalty risk eliminated (Turkish fines can reach $1M for KVKK violations).**
