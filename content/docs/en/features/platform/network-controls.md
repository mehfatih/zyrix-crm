# Network-Level Controls

**What it does:** Platform-wide protection: geo-blocking (block countries you don't do business with), rate limiting (stop brute-force attacks), DDoS heuristics (detect and mitigate attacks). Invisible but critical.

## Key Benefits

- **Block threat origins:** Prevent access from known bad-actor countries
- **Brute force protection:** 100 failed logins → IP blocked for 24 hours
- **DDoS mitigation:** Automatic traffic pattern analysis
- **Always on:** No merchant configuration needed
