# Slack & Microsoft Teams Deep Integration

**What it does:** Notifications, two-way sync, slash commands (/zyrix create deal "Acme Corp"), customer record previews in chat, @mention alerts. Bring CRM into your team's chat flow.

## Key Benefits

- **Zero context switching:** Execute CRM tasks from chat
- **Real-time deal alerts:** "New $50K deal created in pipeline"
- **Customer context on hover:** Mention a customer, see their summary
