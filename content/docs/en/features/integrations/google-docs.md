# Google Docs Cataloging

**What it does:** Link Google Drive documents to CRM records (customers, deals, contracts). Zyrix indexes document content for search. Never lose track of which Drive folder has the MSA for which client.

## Key Benefits

- **Organized document chaos:** Drive stays organized through CRM context
- **Searchable:** Find "pricing proposal for Alpha Corp" across all docs
- **Team handoffs smooth:** New team members see linked docs in context
- **Weekly re-indexing:** Catches document updates
