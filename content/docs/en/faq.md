# FAQ

### Can we migrate from our current CRM?
Yes. Export a CSV from Salesforce, HubSpot, Zoho, or any spreadsheet tool, then use the import wizard.

### Is our data secure?
All data is encrypted at rest and in transit. Enterprise plan adds IP allowlisting, SCIM, and custom retention.

### What's the uptime SLA?
99.9% on the Business plan, 99.99% on Enterprise.

### Do you integrate with our existing tools?
Yes — Slack, Microsoft Teams, Google Docs, Zapier, and any tool via the public REST API and webhooks.

### What happens if we want to leave?
Your data belongs to you. Export every record to CSV or call the Compliance API (GDPR export) at any time.
