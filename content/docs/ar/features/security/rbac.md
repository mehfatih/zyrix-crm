# التحكم بالوصول حسب الأدوار (RBAC)

**ما يفعله:** أنشئ أدواراً مخصصة بأذونات دقيقة. مثال: "مشاهد المبيعات" يمكنه رؤية بيانات العميل والإيرادات لكن لا يمكنه التحرير أو الحذف. "مدير المالية" يمكنه رؤية كل البيانات المالية لكن ليس تواصل العملاء. ٢٥ إذناً دقيقاً عبر الوحدات.

## مثال من الواقع

**مثال حقيقي: "شركة محاسبة جدة"**

شركة محاسبة من ٤٠ شخصاً واجهت مشكلة وصول المحاسبين الصغار لبيانات إيرادات عملاء الشركاء الكبار. غادر محاسب صغير للمنافس بقائمة العملاء ذوي القيمة العالية.

مع RBAC، حصل الصغار على أدوار مخصصة تُخفي أعمدة الإيرادات تماماً. فقط الشركاء يرون الإيرادات. **تم القضاء على الحوادث الأمنية. تحقق الامتثال لمعايير حماية البيانات ISCA/SOCPA.**
