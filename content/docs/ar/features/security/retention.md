# سياسات الاحتفاظ بالبيانات المخصصة

**ما تفعله:** حدد كم من الوقت تُحفظ أنواع بيانات مختلفة. مثال: سجلات التدقيق لـ ٧ سنوات، سجلات العملاء المحذوفة لـ ٣٠ يوماً ثم تُمسح، الرسائل لسنة. تنظيف تلقائي. تجاوز الإيقاف القانوني للتقاضي.

## مثال من الواقع

**مثال حقيقي: "شبكة الرعاية الصحية إسطنبول"**

قانون KVKK التركي (قانون البيانات الشخصية) يتطلب حذف بيانات المرضى بعد X يوم من إغلاق الحساب. تتبع هذا يدوياً عبر ١٤٠,٠٠٠ سجل مريض كان مستحيلاً.

مع احتفاظ البيانات في زيركس، تعمل السياسات تلقائياً: حسابات المرضى المغلقة → PII يُمسح بعد ١٨٠ يوماً. **تم أتمتة امتثال KVKK. تم القضاء على خطر الغرامات (غرامات تركيا قد تصل لـ ١ مليون دولار لانتهاكات KVKK).**
