# قائمة IP البيضاء

**ما تفعله:** قيّد الوصول لـ CRM لعناوين IP محددة أو نطاقات (تدوين CIDR). اقفل النظام لشبكة مكتبك، VPN، أو عناوين منازل موظفين محددين. أي محاولة من خارجها = محجوبة فوراً.

## مثال من الواقع

**مثال حقيقي: "شركة خدمات مالية دبي"**

شركة منظمة من DFSA تطلبت شهادة SOC 2 Type II لعقد حكومي. طلب المدققون قيود IP. كانوا يستخدمون CRM منافس لا يقدم هذا.

التحول لـ زيركس Enterprise مع قائمة IP البيضاء حقق الشهادة. **قيمة العقد: ٢.٤ مليون دولار سنوياً.** وحدها ميزة قائمة IP بررت التحول.
