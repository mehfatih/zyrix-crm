# ضوابط على مستوى الشبكة

**ما تفعله:** حماية على مستوى المنصة: حظر جغرافي (حظر البلدان التي لا تعمل معها)، تحديد معدل (إيقاف هجمات القوة الغاشمة)، استكشاف DDoS (الكشف والتخفيف). غير مرئية لكن حرجة.
