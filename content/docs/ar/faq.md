# الأسئلة الشائعة

### هل يمكن ترحيل البيانات من نظام CRM الحالي؟
نعم. صدّر ملف CSV من Salesforce أو HubSpot أو Zoho أو أي جدول، ثم استخدم معالج الاستيراد.

### هل بياناتنا آمنة؟
جميع البيانات مشفّرة أثناء التخزين والنقل. الخطة المؤسسية تضيف IP Allowlist وSCIM وسياسات الاحتفاظ المخصّصة.

### ما هو SLA الجاهزية؟
99.9٪ على خطة الأعمال، و99.99٪ على الخطة المؤسسية.

### هل تتكاملون مع أدواتنا الحالية؟
نعم — Slack وMicrosoft Teams وGoogle Docs وZapier، وأي أداة عبر واجهة REST العامة وwebhooks.

### ماذا لو أردنا المغادرة؟
البيانات ملكك. صدّرها كـCSV أو استخدم Compliance API للحصول على نسخة كاملة وفق GDPR.
