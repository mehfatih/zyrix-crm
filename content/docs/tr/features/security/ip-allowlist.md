# IP İzin Listesi

**Ne yapar:** CRM erişimini belirli IP adresleri veya aralıklarıyla (CIDR gösterimi) kısıtlayın. Sistemi ofis ağınıza, VPN'inize veya belirli çalışan ev adreslerine kilitleyin. Dışarıdan herhangi bir girişim = anında engellenir.

## Gerçek Hayattan Örnek

**Gerçek Dünya Örneği: "İstanbul Finansal Hizmetler Firması"**

BDDK denetimli bir firma, bir devlet sözleşmesi için SOC 2 Type II sertifikası gerektiriyordu. Denetçiler IP kısıtlamaları istedi. Bunu sunmayan bir rakip CRM kullanıyorlardı.

Zyrix Enterprise'a IP izin listesi ile geçmek sertifikayı kazandırdı. **Sözleşme değeri: yılda 50 milyon TL.** Tek başına IP izin listesi özelliği geçişi haklı çıkardı.
